# Timing and Temperature Reference

## Germination temperatures

| Crop family | Optimum | Minimum | Days to germination |
|-------------|---------|---------|---------------------|
| Tomatoes | 22-25°C | 18°C | 5-10 |
| Peppers / chili | 25-28°C | 20°C | 14-21 |
| Basil | 22-25°C | 20°C | 5-10 |
| Cucumbers / squash | 25-30°C | 18°C | 3-7 |
| Lettuce / greens | 15-20°C | 10°C | 3-7 |
| Herbs (dill, parsley) | 18-22°C | 12°C | 7-21 |
| Strawberries (from seed) | 18-22°C | 15°C | 14-28 |

## Seedling temperatures (after germination)

| Crop family | Day | Night | Grow light hours |
|-------------|-----|-------|-----------------|
| Tomatoes | 20-22°C | 16-18°C | 14-16 |
| Peppers / chili | 22-24°C | 18-20°C | 14-16 |
| Basil | 20-22°C | 18°C+ | 14-16 |
| Cucumbers / squash | 22-25°C | 18-20°C | 14-16 |
| Lettuce / greens | 15-18°C | 12-15°C | 12-14 |
| Herbs | 18-20°C | 15-18°C | 12-14 |

## Sowing lead times (weeks before last frost)

| Crop | Weeks before last frost | Notes |
|------|------------------------|-------|
| Peppers / chili (hot, long season) | 10-12 | Habanero, Bird's Eye — need early start |
| Peppers / chili (early varieties) | 8-10 | Jalapeño, sweet peppers |
| Tomatoes (determinate, compact) | 8-10 | Slower growth, need more lead time |
| Tomatoes (indeterminate, vigorous) | 6-8 | Fast growers, later sow is fine |
| Basil | 6-8 | Very light-sensitive, needs grow light |
| Cucumbers | 3-4 | Fast growers, don't start too early |
| Lettuce / greens | 4-6 | Can direct sow after last frost too |
| Herbs (dill, parsley, cilantro) | 4-6 | Parsley is slow to germinate |

## Transplant timing

| Crop | When to transplant outdoors | Min night temp |
|------|---------------------------|----------------|
| Tomatoes | last_frost + 1-2 weeks | 10°C |
| Peppers / chili | last_frost + 1-2 weeks | 12°C |
| Basil | last_frost + 2 weeks | 12°C |
| Cucumbers / squash | last_frost + 2 weeks | 12°C |
| Lettuce / greens | last_frost - 2-4 weeks | 5°C |
| Herbs (hardy) | last_frost - 2 weeks | 5°C |

## Hardening schedule (universal)

2 weeks before planned transplant date:

| Days | Duration outdoors | Conditions |
|------|------------------|------------|
| 1-3 | 1-2 hours | Sheltered, no direct sun, no wind |
| 4-7 | 3-5 hours | Diffused sunlight allowed |
| 8-11 | Half day | Open air, some direct sun |
| 12-14 | Full day, bring in at night | Full conditions |
| After | Permanent outdoor placement | Only if night temps are safe |

Never put out if forecast night temperature is below: 10°C (tomatoes, peppers), 12°C (basil), 5°C (lettuce, hardy herbs).

## Grow light parameters

| Parameter | Value |
|-----------|-------|
| Type | LED, full spectrum or blue/cool-dominant |
| Height above plants | 30-40 cm |
| Daily duration | 14-16 hours |
| Start | Immediately at first sprout emergence |
| Adjustment | Raise lamp as plants grow; if leggy → lower lamp or add hours |

## Pricking out (transplanting seedlings)

| Crop | When | Container |
|------|------|-----------|
| Tomatoes | 2 true leaves | 7-10 cm pots |
| Peppers | 2-4 true leaves | 7-10 cm pots |
| Basil | 2 true leaves | 7-10 cm pots or groups |
